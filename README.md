# Smart City IoT & Legal AI Suite

A full-stack dashboard application that brings together real-time smart infrastructure management and automated corporate governance. This repository features an asynchronous IoT parking grid simulation engine coupled with an AI-driven legal contract risk assessment protocol.

## 🚀 Features

### 🚗 1. Real-Time Smart Parking Grid
* **Asynchronous Telemetry:** Features a Python-based async background event engine that simulates physical IoT sensor state changes.
* **Proximity Routing:** Dynamically calculates distances and tags bay types (Standard, EV Charging, Handicap) to mimic live driver-routing logic.

### ⚖️ 2. AI Contract Review Suite
* **Automated Risk Profiling:** Mimics an AI review workflow by scanning legal agreements for non-compliant clauses (e.g., notice periods, unlimited liability).
* **Compliance Recommendations:** Automatically flags high-risk sections against a predefined regulatory playbook and suggests remediation text.

### ⚙️ 3. Full Administrative Control Suite
* **Dynamic Infrastructure Provisioning:** Allows administrators to register and deploy new parking sensors onto the live grid on-the-fly.
* **Live Policy Updates:** Features real-time modifications to backend legal risk playbooks directly through the client interface without system downtime.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript (ES6+), State Management (`useState`, `useEffect`), Fetch API, CSS3 Grid Layout
* **Backend:** Python 3, FastAPI, Uvicorn (Asynchronous Web Server), Pydantic (Data Validation & Schemas), Asyncio
